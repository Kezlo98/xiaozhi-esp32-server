# Project Introduction

manager-api is a project developed based on the SpringBoot framework.

When using a code editor for development, select the `manager-api` folder as the project directory when importing.

# Development Environment
JDK 21
Maven 3.8+
MySQL 8.0+
Redis 5.0+
Vue 3.x

# API Documentation
